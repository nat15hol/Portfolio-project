Portfolio Website

This is a portfolio project built with HTML, CSS, and JavaScript. The website is used to showcase projects, experience, and links to GitHub repositories.

🌐 Pages

The website consists of the following pages:

Home – Landing page with a short introduction
About – Information about me/us
CV – Experience, education, and skills
Services – Services we offer
Portfolio – Projects with links to GitHub

🛠️ Technologies
HTML5
CSS3
JavaScript (vanilla)

📁 Structure

The project is structured with separate HTML files for each section, along with shared styling and JavaScript for interactivity.

🚀 Purpose

The purpose of this project is to create a modern and simple portfolio website to showcase projects and technical skills.
